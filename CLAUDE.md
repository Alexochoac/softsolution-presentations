# CLAUDE.md — Presentation Builder Softsolution

## What This Project Does
A Node.js build system that assembles customizable HTML sales presentations
for Softsolution & LiteSentry products, targeted at glass industry customers.

## Tech Stack
- Node.js (build scripts)
- HTML / CSS / JavaScript (slides)
- Customer config files (colors, logos, content)

## Project Structure
(Update this as the structure is built out)
- `slides/` — reusable slide library
- `customers/` — per-customer config (colors, logos, content)
- `build/` or `dist/` — generated presentations
- `scripts/` — Node.js build tooling
